### Directory Seed

This is where you have a directory filled with JSON files that were created from a ```mongoexport```.

__NOTE:__ make sure to use --jsonArray when exporting data from tables. Similar ```mongoexport --db PetShop --collection Food --
out Food.json --jsonArray```

```javascript
mongoSeed.load("localhost",27017, "<name_of_database>", &#
x22;<seed_directory>", "dir", function (err) {
  //..do what ever you need
});
```
